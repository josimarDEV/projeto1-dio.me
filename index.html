Básico que precisamos saber h2

voltar

O HTML, que significa Linguagem de Marcação de Hipertexto, é fundamental para a criação de páginas web. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação compõe-se de marcações que indicam ao servidor da web o estilo e a estrutura do documento.

O HTML não é considerado uma linguagem de programação, pois não pode criar funcionalidades dinâmicas. É amplamente utilizado em Desenvolvimento Web, Navegação na Internet e Documentação.

Como estruturar a base do HTML para criar um site. h3

Para começar a criar um site com HTML, é essencial entender a estrutura básica. Dentro do elemento HTML, temos o head (cabeçalho) que contém informações sobre o documento, e o body (corpo) onde o código será estruturado. Vamos explorar as tags que detalham nosso site.

O que são TAGS? h3

Tags HTML são instruções que informam ao navegador como formatar o texto. Elas podem ser usadas para definir vários parâmetros, como itálico, quebras de linha, objetos, bullet points e muito mais.

Essas tags são lidas pelos servidores na ordem de cima para baixo, semelhante à redação de um texto. Podemos utilizar quantas tags quisermos para formatar conteúdos, mas é essencial seguir algumas regras básicas:

Uma tag HTML deve conter três partes: h4

  1. Uma tag de abertura -- começando com um símbolo <.
  2. Conteúdo -- instruções curtas para exibir o elemento na página.
  3. Uma tag de fechamento -- terminando com um símbolo </.

5 Exemplos de tags: h4

  1. <p> texto </p>: Tag de parágrafo, formata qualquer texto entre a tag de abertura e de fechamento, como um parágrafo padrão ou corpo principal.
  2. <b> texto </b>: Tag de negrito, formata qualquer texto entre as tags como negrito.
  3. <h2> texto </h2>: Tag de título, formata qualquer texto entre as tags. Existem de h1 até h4.
  4. <i> texto </i>: Tag itálico, formata qualquer texto entre as tags como itálico.
  5. <u> texto </u>: Tag de sublinhado, formata todo texto entre as tags como sublinhado.

Links e URLs h3

voltar

Os links são elementos fundamentais para a navegação e interconexão de páginas web. Eles são definidos pelo elemento <a> (âncora) e podem conter atributos como href para especificar o destino do link. Abaixo estão alguns pontos-chave sobre links:

  • Elemento <a>: Utilize o seguinte formato para criar um link: 
    <a href="url_do_destino">Texto do Link</a>
  • Atributo href: Especifica o URL para o qual o link aponta.
  • Links internos: Use caminhos relativos para vincular páginas dentro do mesmo site.
  • Links externos: Coloque o URL completo para vincular páginas externas.

Listas ordenadas e não ordenadas h3

voltar

As listas são fundamentais para organizar informações. No HTML, você pode criar listas ordenadas usando o elemento <ol> e listas não ordenadas usando <ul>. Cada item da lista é representado por <li>. Abaixo estão alguns pontos importantes sobre listas:

  • Listas ordenadas (ol): Utilize o elemento <ol> para criar listas numeradas.
  • Listas não ordenadas (ul): Utilize o elemento <ul> para criar listas sem números.
  • Itens de lista (li): Cada item da lista é representado por <li>.
  • Nesting: Você pode aninhar listas, colocando uma lista dentro de outra.

Imagens e mídia h3

voltar

O elemento <img> é usado para incorporar imagens em uma página web. Além disso, você pode incorporar outros tipos de mídia, como áudio e vídeo. Abaixo estão alguns pontos-chave sobre imagens e mídia:

  • Elemento <img>: Use o atributo src para especificar o caminho da imagem.
  • Alt: Sempre forneça um texto alternativo usando o atributo alt para acessibilidade.
  • Mídia externa: Use elementos como <audio> e <video> para incorporar áudio e vídeo.

Tabelas para organização de dados h3

voltar

O elemento <table> é usado para criar tabelas no HTML. Tabelas são úteis para organizar dados em linhas e colunas. Abaixo estão alguns pontos importantes sobre tabelas:

  • Elemento <table>: Use <table> para iniciar uma tabela.
  • Linhas e colunas: Utilize <tr> para criar uma linha e <td> para células de dados.
  • Cabeçalhos de tabela: Use <th> para criar cabeçalhos de coluna ou linha.
  • Estilo: Aplique estilos com CSS para melhorar a aparência da tabela.

Campos de texto e áreas de texto h3

voltar

Campos de texto e áreas de texto permitem que os usuários insiram informações. Use o elemento <input> para campos de texto e <textarea> para áreas de texto.

  • Elemento <input>: Use o atributo type para especificar o tipo de campo, como texto, senha, e-mail, etc. 
    <input type="text" name="nome" id="nome" placeholder="Seu Nome">
  • Elemento <textarea>: Crie áreas de texto para entrada de dados mais longos. 
    <textarea name="comentario" id="comentario" rows="4" cols="50" placeholder="Digite seu comentário"></textarea>

Botões e seletores h3

voltar

Botões e seletores são essenciais para a interatividade do formulário. Utilize os elementos <button> e <select> para esses fins.

  • Elemento <button>: Crie botões clicáveis. 
    <button type="submit">Enviar</button>
  • Elemento <select>: Crie listas suspensas (dropdown) para seleção. 
    
        <select name="carro" id="carro">
            <option value="volvo">Volvo</option>
            <option value="saab">Saab</option>
            <option value="mercedes">Mercedes</option>
            <option value="audi">Audi</option>
        </select>

Validação de formulários com atributos h3

voltar

Os atributos podem ser usados para validar dados do formulário, garantindo que os usuários insiram informações corretas.

  • Atributo required: Torna um campo obrigatório. 
    <input type="text" name="obrigatorio" id="obrigatorio" required>
  • Atributo pattern: Especifica um padrão para a entrada. 
    <input type="text" name="padrao" id="padrao" pattern="[0-9]{3}" placeholder="Apenas números de 3 dígitos">

Enviando dados com o elemento <form> h3

voltar

O elemento <form> é usado para agrupar elementos de formulário e enviar dados para um servidor. Use o atributo action para especificar o URL de destino.

 
        <form action="/processar-formulario" method="post">
            <!-- Seus elementos de formulário aqui -->
            <button type="submit">Enviar</button>
        </form>
